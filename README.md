# kotlin-cli
## Todo List Cli Project


### 기능 개요
Kotlin으로 간단한 CLI 기반 TodoList를 구현합니다.
사용자는 오늘 해야 할 일을 등록하고, 완료 여부를 관리할 수 있습니다.
이후 확장 단계에서는 날짜별 일정 관리가 가능한 “캘린더형 투두리스트”로 발전시킬 예정입니다.


#### 아이디어
당일 해야할 목록을 작성하는 TodoList를 구현한다.
사용자에게 번호와 해야할 내용을 입력받는다.

이렇게 입력받은 내용을 바탕으로 데이터를 저장한다.
해당 프로젝트에는 3가지 기능을 구현할 목적이 있는데,

1. 해야할 작업을 리스트에 추가하는 기능 
2. 끝난 작업을 리스트에서 지우는 기능
3. 리스트에 있는 작업들을 반환하는 기능


#### 구현 목록
무한 루프를 돌며 1 입력시에 프로그램이 종료되도록 한다.
add 명령어를 이용하여 리스트에 작업할 내용을 추가한다.
done 명령어를 이용하여 작업한 내용을 지운다.
list 명령어를 이용하여 작업 리스트를 반환한다.

- 입력한 순서대로 리스트에 담아 저장한다. (1. ~, 2.~)
- done을 입력하면 어떤 리스트를 지울건지 물어보고 번호를 입력받는다.

----
(ex)
명령어를 입력하세요!(add, done, list, exit)
> add
> 작업할 내용을 입력하세요 : 
> "Kotlin 공부하기"
> 저장되었습니다! 

명령어를 입력하세요!(add, done, list, exit)
> add 
> 작업할 내용을 입력하세요 : 
> "헬스장 가기"
> 저장되었습니다!

명령어를 입력하세요!(add, done, list, exit)
>done
> 완료한 작업을 선택해주세요.(번호만 입력)
    [목록]
    1. Kotlin 공부하기
    2. 헬스장 가기
> 1
> 고생하셨습니다!

명령어를 입력하세요!(add, done, list, exit)
>list
    [목록]
    1. 헬스장 가기
> 오늘 목표를 향해 조금만 더 노력하세요!

명령어를 입력하세요!(add, done, list, exit)
>exit
> 프로그램을 종료합니다.

#### 에러처리
1. 기능에 없는 명령어를 입력할 경우
2. 똑같은 작업 내용을 입력한 경우
3. 완료한 작업에서 입력한 번호가 없는 경우, null값인 경우


##### 추가적으로 구현하고 싶은 기능
